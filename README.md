# アポ管理ツール

新歓向けのアポ管理 Web アプリです。主機能は「誰の担当の新入生が、いつ来るか」を日付ベースで管理することです。β機能として、3月から5月の担当者シフト入力もできます。

現在は Supabase 同期型です。ブラウザごとの `localStorage` ではなく、共有DBに保存します。
認証ログインは使わず、Supabase の公開キーと RLS の `anon` ロールでアクセスします。

## できること

- 新歓相手の登録、編集、論理削除
- 氏名 + 大学・学部ベースの重複検知
- 日付ごとの一覧表示と検索
- 画面上で担当者名を選ぶ簡易本人識別
- 担当者名の追加と既存名選択
- 3月、4月、5月のシフト入力
- 日別の昼参加可能者一覧、夜参加可能者一覧、未入力者一覧

## 起動

```bash
npm start
```

ブラウザで [http://localhost:4173](http://localhost:4173) を開いてください。

## テスト

```bash
npm test
```

## Supabase セットアップ

初回だけ、Supabase の SQL Editor で `supabase/schema.sql` の内容を実行してください。

必要なテーブル:

- `members`
- `appointments`
- `shifts`

テーブル未作成のまま起動すると、画面上にセットアップ不足のエラーが表示されます。
